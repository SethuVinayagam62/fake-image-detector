# AI Fake Image Detection

## Project Description

This project detects whether an image is real or AI-generated using a deep learning model.
The system allows users to upload an image and returns the percentage of AI influence in the image.

## Features

* Upload image for analysis
* Detect AI-generated images
* Display prediction percentage
* Simple web interface

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* Flask
* HTML
* CSS

## Project Structure

app.py – Main Flask application
train_model.py – Model training code
templates/index.html – Web page for image upload
model.h5 – Trained deep learning model

## How to Run the Project

1. Install required libraries
   pip install -r requirements.txt

2. Run the application
   python app.py

3. Open browser and go to
   http://127.0.0.1:5000/
